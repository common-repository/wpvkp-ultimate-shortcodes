WPVKP ultimate shortcodes plugin helps you to add modern and responsive call to action buttons, Google maps, social media share buttons, alert boxes and many other elements to make your website look more professional and attractive.

== Description ==

WPVKP Ultimate Shortcodes is easy to use wordpress shortcode plugin which works with almost every wordpress theme and with Theme Frameworks like Genesis or Thesis child themes.

I have developed this plugin so that I could recommend a powerful and regularly updated plugin to my <a rel="dofollow" href="http://wpvkp.com/best-wordpress-magazine-themes/" target="_blank">wordpress magazine theme</a> users.

= Unique Features =

1. Modern & attractive buttons.
2. Mobile responsive elements like columns, alert-boxes, Google maps, etc.
3. Integrated social media icons.
4. Support for Font Awesome.
5. Support for fontello.
6. Retina ready icons which makes your website look professional on every device.
7. Three different types of download buttons.
8. Unlimited color variations available.
9. Cool CSS3 transition effects.
10. Support for tabs.
11. Support for accordions.
12. One click add youtube videos.
13. Integrated toggles with option to choose icons from fontawesome.

== Installation ==

Installing this plugin is really simple and straightforward.

1. Move to your wordpress administration panel.
2. Navigate to "Plugins".
3. Click on "Add New" button on top of the page.
4. Search for WPVKP Ultimate Shortcodes.
5. Install this plugin.
6. Activate the plugin.
7. Add new post or edit your previous post and you will find a new incon with "W" in your editor.
8. Click on "W" icon and add the elements you want.

Enjoy, and please don't forget to share this plugin and to rate this plugin, if you liked it.
